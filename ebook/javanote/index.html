笔记将全面使用 Java 8 句法，以前可能使用匿名嵌套类的地方会换用 lambda 表达式。

为什么写这套笔记

学习使用 Java 至今已经有几年时间了，期间除了学校教授的内容之外，在网上也学习了很多更深入的内容。

小昇一直有做笔记的习惯，基本每学习一门新课程，就会有一本对应的笔记。在初学 Java 语言时，小昇也整理摘要了许多内容，不过一直都比较杂乱，没有系统的框架，而且学校教授的内容早已过时，并不能反映 Java 的很多新特性。

这一次，在参考《Java 技术手册（第6版）》的基础上，小昇决定重新整理一下 Java 的基础知识。

面向读者

因为是笔记，所以主要面向的读者是有一定编程基础，或者是正在学习 Java 的人群。当然，即使是已经熟练运用 Java 的人士也可以当做参考，便于在编程时查阅一些细节问题。

小昇也充分考虑到初学者的情况，所以基础知识也会占用很大的篇幅，即使你没有任何编程基础也可以将本套笔记作为入门的参考材料。
