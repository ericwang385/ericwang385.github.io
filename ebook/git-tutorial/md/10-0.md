# 期末总结

终于到了期末总结的时刻了！

经过几天的学习，相信你对 Git 已经初步掌握。一开始，可能觉得 Git 上手比较困难，尤其是已经熟悉 SVN 的童鞋，没关系，多操练几次，就会越用越顺手。

Git 虽然极其强大，命令繁多，但常用的就那么十来个，掌握好这十几个常用命令，你已经可以得心应手地使用 Git 了。

友情附赠国外网友制作的 Git Cheat Sheet，建议打印出来备用：

[Git Cheat Sheet](http://pan.baidu.com/s/1jGxjQL4#path=%252Fgit)

现在告诉你 Git 的官方网站：[http://git-scm.com](http://git-scm.com/)，英文自我感觉不错的童鞋，可以经常去官网看看，而且现在官方提供的[参考书](https://git-scm.com/book/zh/)已经有中文版了。什么，打不开网站？相信我，我给出的绝对是官网地址，而且，Git 官网决没有那么容易宕机，可能是你的人品问题，赶紧面壁思过，好好想想原因。

如果你学了 Git 后，工作效率大增，有更多的空闲时间健身看电影，那我的教学目标就达到了。

谢谢观看！
